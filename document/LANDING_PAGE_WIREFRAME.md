# Landing Page Wireframe (Single-Language Campaign Page)

This block order is designed for paid traffic and high-intent visitors.

## Block Order

1. **Hero (Above the fold)**
   - Headline, subheadline, 1 primary CTA, 1 secondary CTA
   - Optional hero background and/or side image

2. **Trust Strip**
   - 3-6 trust badges (years, output scale, client count, turnaround, quality controls)

3. **Capabilities**
   - "Why we are strong" cards
   - Focus on operational reliability, quality control, and speed

4. **Product Gallery (Grid/Masonry, not featured slider)**
   - Image-first product categories
   - 8-12 product tiles with links to quote flow or product pages

5. **Proof / Outcome Snapshot**
   - 3 compact case metrics with business outcomes
   - One short quote each

6. **Process (4 steps)**
   - Consult -> Prepress -> Production -> Delivery
   - Keep copy short and visual

7. **FAQ**
   - Conversion-focused questions (turnaround, MOQ, file formats, delivery, support)

8. **Final CTA**
   - Restate offer, include urgency/reliability angle
   - Primary CTA + phone CTA

## Language Rollout

- Phase 1: Spanish (`/lp/es`)
- Phase 2: Chinese (`/lp/zh`) clone structure
- Phase 3: Hebrew (`/lp/he`) clone structure with RTL enabled

## RTL Notes (Hebrew)

- Set `dir="rtl"` at page root for Hebrew.
- Mirror horizontal spacing/alignment and icon directions where needed.
- Keep numbers and metrics readable by preserving numeric glyph order.
